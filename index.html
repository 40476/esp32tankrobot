Tank Robot Commands

TANK CONTROL (with PWM speed control -200 to +200)

  • ts - Stop tank (sent automatically when sliders released)
  • tms <L> <R> - Set tank motors separately (supports negative values!)
  • Examples: tms 150 -100, tms-50,100, tms -200 200
  • Tank sliders: Hold to move, release to stop
  • Note: Positive values = forward, negative values = reverse

CLAW & REACH (always full speed)

  • Hold buttons to run, release to stop
  • m0f / m0b - Claw Grab forward/backward
  • m1f / m1b - Claw Rotate forward/backward
  • m2f / m2b - Middle Reach forward/backward
  • m3f / m3b - Base Reach forward/backward

GLOBAL COMMANDS

  • s - Emergency stop all motors
  • h - Show this help
  • d - Run demo sequence
  • status - Print current status
⚠️ Safety Notes:
• Robot will STOP IMMEDIATELY if Bluetooth disconnects
• Different buzzer tones indicate connection status
• Keep robot in visible area while controlling
• Emergency stop button stops all motors immediately

Note: Commands must be terminated with newline (automatic).

Compatibility: Requires Chrome/Edge browser with Web Bluetooth support (HTTPS or localhost only)
