Tank Robot Commands

TANK CONTROL (with PWM speed control 0-200)

  • ts - Stop tank (should be called every 2 seconds when tank sliders are at 0)
  • tms<L><R> - Set tank motors separately (e.g., tms150-100)
  • Note: Speeds 0-200. Negative values = reverse.

CLAW & REACH (always full speed)

  • m0f - Claw Grab forward
  • m0b - Claw Grab backward
  • m0s - Claw Grab stop
  • m1f - Claw Rotate forward
  • m1s - Claw rotate stop
  • m1b - Claw Rotate backward
  • m2f - Middle Reach forward
  • m2s - Middle Reach stop
  • m2b - Middle Reach backward
  • m3f - Base Reach forward
  • m3s - Base Reach stop
  • m3b - Base Reach backward

GLOBAL COMMANDS

  • s - Stop all motors
  • h - Show this help

Note: Commands must be terminated with a newline character (automatically added by this interface).
